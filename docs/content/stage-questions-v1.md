**Status:** PM + Care Specialist sign-off is linked from the PR that lands TASK-034. Until then, this file tracks the same strings as `apps/web/src/lib/onboarding/questions-v1.ts`.

## Rationale (engineering placeholder)

- Plain-language, behavioral questions avoid clinical labels shown to the caregiver.
- Ordinals and multi-select increase fidelity versus yes/no/unsure while keeping the flow to eight questions.

## The eight questions (v1)

1. **Medications** — self / reminders / hands-on help  
2. **Driving** — safe, worried, stopped (recent / long ago), never drove  
3. **Alone for a few hours** — multi-select chips (incl. “nothing in particular”)  
4. **Recognition** — always, usually, sometimes, rarely  
5. **Bathing & dressing** — on own, with reminders, hands-on help  
6. **Wandering / lost in the last year** — no, once, a few times, often  
7. **Conversation** — full exchange, repetition, only short, rarely makes sense  
8. **Sleep** — through the night, some nights hard, most nights hard
